Что выведет программа?

```go
package main
 
import (
    "fmt"
)
 
func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```
Объяснить вывод программы.

```
Программа выведет [77 78 79]

Создается массив a из пяти целых чисел.
Затем создается срез b, который является подмножеством массива a, начиная с индекса 1 (включительно) и заканчивая индексом 4 (не включительно)
При выводе среза b с помощью fmt.Println(b), получается [77 78 79], потому что это элементы массива a на позициях 1, 2 и 3
```
