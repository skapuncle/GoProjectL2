Что выведет программа?

```go
package main
 
import (
    "fmt"
)
 
func test() (x int) {
    defer func() {
        x++
    }()
    x = 1
    return
}
 
 
func anotherTest() int {
    var x int
    defer func() {
        x++
    }()
    x = 1
    return x
}
 
 
func main() {
    fmt.Println(test())
    fmt.Println(anotherTest())
}
```

Объяснить вывод программы. Объяснить как работают defer’ы и порядок их вызовов.

```
В функции test:
Объявляетя функция test с именованным возвращаемым значением x.
Внутри функции test используется defer для объявления анонимной функции, которая будет увеличивать x на 1 после того, как основная функция test завершит свою работу.
Затем присваивается x значение 1.
При возврате из функции test, анонимная функция, объявленная с defer, выполняется, увеличивая x на 1, и, следовательно, test возвращает 2.

В функции anotherTest:
Объявляется функция anotherTest с локальной переменной x.
Внутри функции anotherTest используется defer для объявления анонимной функции, которая будет увеличивать x на 1 после того, как основная функция anotherTest завершит свою работу.
Затем присваивается x значение 1.
При возврате из функции anotherTest, анонимная функция, объявленная с defer, выполняется, увеличивая x на 1.
Однако, поскольку x объявлен внутри функции anotherTest и не является именованным возвращаемым значением, изменение x внутри анонимной функции defer не влияет на возвращаемое значение anotherTest.
Таким образом, anotherTest возвращает 1, поскольку это значение x до выполнения defer
```
